# Codex Web App

Cette application simple intègre l'API OpenAI Codex pour générer du code.

## Installation

1. Clonez le projet  
2. Copiez `.env.example` en `.env` et ajoutez votre clé API OpenAI.  
3. Installez les dépendances :
```
npm install
```
4. Lancez le serveur :
```
npm start
```
5. Ouvrez votre navigateur à `http://localhost:3000`

## Utilisation
Entrez un prompt et cliquez sur "Générer le code".
